Körper

Ein Körper besteht aus einer Menge K von Elementen zusammen mit zwei Verknüpfungen + und * die je zwei Elementen x, y ∈ K wieder ein Element x + y bzw. x * y von K zuordnen.

Damit eine solche Struktur Körper genannt wird, müssen die folgenden drei Gruppen von Gesetzen (Axiome) für alle x, y, z ∈ K erfüllt sein:

Axiome

Assoziativität
(Klammern sind beliebig)

(x + y) + z = x + (y + z)

Nullelement (Addition only)
Es gibt ein Element 0 ∈ K, sodass x + 0 = x für alle x ∈ K

Einselement (Multiplikation only)
Es gibt ein Element 1 ∈ K, sodass x * 1 = x für alle x ∈ K

Inverse
Zu jedem x ∈ K gibt es ein Element -x ∈ K, sodass x + (-x) = 0

Kommutativität

x + y = y + x

Distributivgesetz

x * (y + z) = x * y + x * z

Vektorräume

Einem Vektorraum liegt immer ein Körper K vor.
Ein K-Vektorraum besteht aus einer Menge V von Elementen, auch Vektoren genannt, die den folgenden Gesetzen genügt:

Verknüpfungen von Vektoren

Es gibt eine Verknüpfung + auf V, die je zwei Vektoren v und w einen Vektor v + w zuordnet, so dass für alle u, v, w ∈ V die folgenden Eigenscha!en erfüllt sind:

Axiome

Assoziativität
(Klammern sind beliebig)

(u + v) + w = u + (v + w)

Nullelement
Es gibt ein Element 0 ∈ V, sodass v + 0 = v für alle v ∈ V

Negative Vektoren
Zu jedem v ∈ V gibt es ein Element -v ∈ V, sodass v + (-v) = 0

Kommutativität

u + v = v + u
