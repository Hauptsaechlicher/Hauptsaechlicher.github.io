Abbildungen

Abbildungen von Menge A und Menge B werden wie folgt dargestellt:

Abbildungen haben dabei folgende Eigenschaften:

Injektiv

Eine Abbildung ist injektiv, wenn jedes Element aus A auf ein Element aus B abgebildet wird.
Zu Zeigen ist dass für alle a1 und a2 aus A gilt: f(a1) ≠ f(a2) => a1 ≠ a2

Surjektiv

Eine Abbildung ist surjektiv, wenn jedes Element aus B mindestens einmal abgebildet wird.
Zu Zeigen ist dass für jedes b aus B ein a aus A existiert, sodass f(a) = b

Körper

Ein Körper besteht aus einer Menge K von Elementen zusammen mit zwei Verknüpfungen + und * die je zwei Elementen x, y ∈ K wieder ein Element x + y bzw. x * y von K zuordnen.

Damit eine solche Struktur Körper genannt wird, müssen die folgenden drei Gruppen von Gesetzen (Axiome) für alle x, y, z ∈ K erfüllt sein:

Axiome

Assoziativität
(Klammern sind beliebig)

(x + y) + z = x + (y + z)

Nullelement (Addition only)
Es gibt ein Element 0 ∈ K, sodass x + 0 = x für alle x ∈ K

Einselement (Multiplikation only)
Es gibt ein Element 1 ∈ K, sodass x * 1 = x für alle x ∈ K

Inverse
Zu jedem x ∈ K gibt es ein Element -x ∈ K, sodass x + (-x) = 0

Kommutativität

x + y = y + x

Distributivgesetz

x * (y + z) = x * y + x * z

Vektorräume

Einem Vektorraum liegt immer ein Körper K vor.
Ein K-Vektorraum besteht aus einer Menge V von Elementen, auch Vektoren genannt, die den folgenden Gesetzen genügt:

Verknüpfungen von Vektoren

Es gibt eine Verknüpfung + auf V, die je zwei Vektoren v und w einen Vektor v + w zuordnet, so dass für alle u, v, w ∈ V die folgenden Axiome erfüllt sind:

Vektoren sind Element von V
Skalare sind Element von K

Axiome

Assoziativität
(Klammern sind beliebig)

(u + v) + w = u + (v + w)

Nullelement
Es gibt ein Element 0 ∈ V, sodass v + 0 = v für alle v ∈ V

Negative Vektoren
Zu jedem v ∈ V gibt es ein Element -v ∈ V, sodass v + (-v) = 0

Kommutativität

u + v = v + u

Unterverktorraum

Ein Untervektorraum U von V ist eine Teilmenge von V, die selbst ein Vektorraum ist.

Kriterien

1. U ist nicht leer (U /= ∅)
2. ∀v, w ∈ U und ∀λ, μ ∈ K gilt:
     λv + μw ∈ U

Zu Zeigen ist dass U für alle u, v ∈ U und alle k ∈ K die folgenden Bedingungen erfüllt:
1. u + v ∈ U
2. k * u ∈ U

Erzeugendensystem

Ein Erzeugendensystem von V ist eine Teilmenge von V, die durch Linearkombinationen
alle Vektoren von V erzeugt.

lR - Vektorraum(lR3, +, *)
E = {e1, e2, ..., en} ⊆ lR3
ist Erzeugendensystem von lR3 wenn gilt:

∀v ∈ lR3:
∃λ1, λ2, λ3 ∈ lR:
v = λ1e1 + λ2e2 + λ3e3

Beispiel:
\[ \begin{bmatrix} x \\ y \\ z \end{bmatrix} = x \begin{bmatrix} 1 \\ 0 \\ 0 \end{bmatrix} * y \begin{bmatrix} 0 \\ 1 \\ 0 \end{bmatrix} * z \begin{bmatrix} 0 \\ 0 \\ 1 \end{bmatrix} \]

Lineare (Un)abhängigkeit

Eine Menge von Vektoren ist linear unabhängig, wenn sich keiner der Vektoren als Linearkombination der anderen Vektoren darstellen lässt.
